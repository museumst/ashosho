📖 JSON 파일을 쉽게 만드는 방법

1

무료 AI 서비스에 접속하세요.
아래 중 하나를 클릭하세요 (모두 무료로 사용 가능합니다).

💬 ChatGPT ✨ Claude 🔷 Gemini
2

아래 질문을 복사하여 AI에게 보내세요.
단어 목록과 단어장 이름을 원하는 대로 바꾸어 사용하세요.

다음 영어 단어들로 단어장 JSON 파일을 만들어줘. 단어장 이름: 나의 영어 단어장 단어: apple, beautiful, change, dream, enjoy, friend, happy, important, kind, learn 아래 형식을 정확히 지켜서 JSON만 출력해줘. 다른 설명은 필요 없어. { "title": "나의 영어 단어장", "entries": [ { "word": "apple", "meaning": "사과", "example": "I eat an apple every morning.", "translation": "나는 매일 아침 사과를 먹는다." } ] }
3

AI가 생성한 내용을 메모장(Windows) 또는 텍스트편집기(Mac)에 붙여넣으세요.
저장할 때 파일명을 단어장.json처럼 .json으로 끝나게 저장하세요.
Windows 메모장: '다른 이름으로 저장' → 파일 형식을 '모든 파일'로 선택 → 파일명 끝에 .json 입력

4

위 업로드 영역에 저장한 파일을 올리세요.
끌어다 놓거나 클릭해서 선택하면 됩니다.
